# Prompt Library

A private, curated library of reusable AI prompts kept for easy access, search, and continuous improvement.

## What's here
- `PROMPTS.md` - the full collection, one prompt per collapsible section (title + ready-to-copy prompt in a fenced block).

## Use
- Browse `PROMPTS.md`, or search with GitHub's file search / `Ctrl+F`, or grep locally:
  ```bash
  grep -i -A3 "translator" PROMPTS.md
  ```

## Improve
- Edit a prompt in place, add new ones following the existing `<details>` pattern, and commit. Treat this as a living library.
